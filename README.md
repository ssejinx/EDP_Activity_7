# Tasked - Task Management System

A Task Management Information System designed for group tasks. It allows users to create tasks, assign members, track status, and generate reports.

---

### ✱ FEATURES

- User Authentication (Login/Logout)
- Password Recovery
- User Management (Add, Edit, Activate, Deactivate Accounts)
- Task Management (Create, Assign, Update Status)
- Report Generation with Excel Export

---

---

### ✱ HOW TO RUN

1. Clone the repository
2. Open `tasked_forms.sln` in Visual Studio
3. Restore NuGet packages
4. Update database connection string
5. Press `F5` to run
